Contribute a Package

1

Create Package

Write your function blocks in IEC 61131-3 ST, create a manifest.json, and package as .ieclib (ZIP).

2

Fork & PR

Fork iec2-registry, add your package to packages/, and submit a Pull Request.

3

Auto Publish

CI validates your package. Once merged, it appears in the registry and VS Code extension automatically.

Install from VS Code

Ctrl+Shift+P → IEC: Manage Libraries → Online → Search & Install
